# ClearPath

**Accessible hazard navigation app for wheelchair users and visually impaired students.**

BeaverHacks 2026 · Social Impact Track · Oregon State University

---

## The Problem

People who use wheelchairs or are visually impaired navigate the same sidewalks every day with zero advance warning of what's ahead. A blocked ramp, a Starship robot parked on a curb cut, cracked pavement, or construction — Google Maps doesn't tell you any of this. These people find out only when they arrive.

## The Solution

ClearPath is a crowdsourced hazard reporting and accessible navigation mobile app built with React Native + Expo. Any student reports a hazard in under 10 seconds. Gemini analyzes it. A pin appears on the live map. Nearby wheelchair and visually impaired users get an automatic voice alert — before they reach the obstacle.

---

## Tech Stack

| Layer | Technology | Purpose |
|---|---|---|
| Mobile app | React Native + Expo | Cross-platform iOS & Android |
| Map | react-native-maps | Campus map with custom hazard pins |
| Navigation | expo-router | Screen routing |
| Road data | Waze API | Real-time road hazards as background layer |
| AI triage | Google Gemini Vision API | Photo → hazard type + urgency + alert text |
| Database | Firebase Firestore | Real-time sync of all reports |
| Backend | Python (Flask) | API server handling Gemini + Firebase |
| Voice | expo-speech | Reads hazard alerts aloud |
| Location | expo-location | GPS tracking and proximity detection |
| Accessibility | React Native Accessibility API | Screen reader support throughout |

## Getting Started

### Prerequisites

- Node.js 18+
- Expo Go app installed on your phone (iOS or Android)

### Setup

```bash
# Clone the repo
git clone https://github.com/your-team/clearpath.git
cd clearpath

# Install dependencies
npm install

# Install Expo-specific packages
npx expo install expo-speech expo-location expo-router react-native-maps

# Start the dev server
npx expo start
```

Scan the QR code with Expo Go on your phone to run the app instantly.

### Backend

```bash
cd backend
pip install flask firebase-admin google-generativeai python-dotenv requests flask-cors
python app.py
```

---

## Environment Variables

Create a `.env` file in `/backend`:

```
GEMINI_API_KEY=your_key_here
```

Get your Gemini API key from [aistudio.google.com](https://aistudio.google.com).

Get your Firebase config from [console.firebase.google.com](https://console.firebase.google.com).

**Never commit `serviceAccountKey.json` or `.env` to GitHub.**

## Data Sources

- **Users** — ADA-specific hazard reports (ramps, robots, cracks) submitted directly in the app
- **Waze API** — road closures, construction, accidents as a background layer
- **OpenSidewalks** — sidewalk width, surface, and slope data for the Seattle/UW area
- **Team** — manually seed 8–10 real OSU campus hazards before judging starts
